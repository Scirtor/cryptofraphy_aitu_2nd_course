# cryptofraphy_aitu_2nd_course

## Overview
This repository contains cryptography course materials and implementations for AITU 2nd year students.

## Contents
- Lecture notes and theoretical concepts
- Algorithm implementations
- Practice exercises and assignments
- Resource references

## Getting Started
1. Clone this repository
2. Review the course materials in order
3. Complete exercises in each section
4. Test implementations with provided test cases

## Prerequisites
- Basic understanding of mathematics
- Programming knowledge (Python/C++)
- Familiarity with number theory concepts
